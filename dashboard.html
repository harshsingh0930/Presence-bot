Setup Guide

Get your bot running in 5 minutes

1 · Create the Bot

  1. Go to discord.com/developers/applications
  2. Click New Application → name it → Save
  3. Go to Bot tab → Add Bot → Copy Token
  4. Enable: Presence Intent, Server Members Intent, Message Content Intent
  5. Go to OAuth2 → URL Generator → select bot + applications.commands
  6. Copy the generated URL and open it to invite the bot to your server

2 · Install & Run

pip install discord.py matplotlib numpy pandas

# Create .env with your token:
echo "DISCORD_TOKEN=YOUR_TOKEN_HERE" > .env

# Run the bot:
python bot.py

3 · Configure in Discord

After the bot joins your server, use:

/setup log_channel:#presence-logs report_channel:#reports

This tells the bot where to post live presence changes and reports.
